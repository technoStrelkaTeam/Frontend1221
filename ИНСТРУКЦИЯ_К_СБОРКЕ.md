
# ИНСТРУКЦИЯ ПО СБОРКЕ КЛИЕНТА

### I. Установите Flutter и зависимости
Подробнее: https://docs.flutter.dev/install
1. Используйте Visual Studio Code. Откройте папку и на запрос от Visual Studio Code установите Flutter.
2. Введите команду в терминал в той же папке
   flutter pub get




### II. Настройка


## Обратите внимание ##
Для запуска web-сборки нужно сменить IP-адрес на `http://127.0.0.1:8000` в файле `\lib\core\network\app_config.dart`


```bash
class AppConfig {
  const AppConfig({
    required this.apiBaseUrl,
  });

  /// Пример: `https://api.company.ru` или `http://10.0.2.2:8080` (Android emulator).
  final String apiBaseUrl;

  static const AppConfig defaults = AppConfig(
    apiBaseUrl: String.fromEnvironment(
      'API_BASE_URL',
      defaultValue: 'http://10.60.63.153:8080',
    ),
  );
}
//http://10.60.63.153:8080/',
```

Для запуска на приложения на другом устройстве укажите локальный IPv4-адрес устройства на котором развёрнут бекенд аналогично инструкции приведённой выше:

Узнать IPv4-адрес можно через команды:
```bash
ip a ## Linux and macos
ipconfig ## Windows
```

### III. Сборка

1. В зависимости от желанной платформы выполните команду

   
   WEB: ```flutter build web```



   Android: ```flutter build apk --release```



   *(!!! УБЕДИТЕСЬ, ЧТО ПО ПУТИ К ПРОЕКТУ ТОЛЬКО ЛАТИНСКИЕ СИМВОЛЫ !!!)*



   IOS: ```flutter build ios```


   
После можете запустить этот файл на вашем устройстве.




В случае в вебом недостаточно просто запустить index.html. Выполните команды в терминале




   ```bash
pip install http.client
   python --http.server 8080
```



   
   В папке с собранным вебов и перейдите по адресу localhost:8080

### IV. Запуск без сборки (тест)
   В терминале в той же папке команда
   ```bash
   flutter run
```

Подключите android-устройство с режимом отладки перед выполнением команды если хотите тестировать android (с ios соответствующе)
